#+title: opencloud-pve — self-hosted cloud & communication
#+PROPERTY: header-args:bash :noeval

* Initial deployment
** Create Proxmox VM

Create VM manually in Proxmox with:
- 2 SCSI disks: sda (32G, OS), sdb (300G, data)
- Attach NixOS minimal ISO to CD drive
- Boot from ISO

** Configure DNS

Configure static DHCP + DNS entries via pfSense:
- opencloud.stfl.dev
- office.stfl.dev

** Partition with disko + install with nixos-anywhere

From the deployment machine (not the VM):

#+begin_src bash
nix run github:nix-community/nixos-anywhere -- \
    --flake '.#opencloud-pve' \
    --phases kexec,disko \
    root@opencloud.stfl.dev
#+end_src

This partitions both disks via disko:
- sda: 1M BIOS boot (EF02) + ext4 root (label: nixos)
- sdb: ext4 data (label: data)

Then install NixOS:

#+begin_src bash
nix run github:nix-community/nixos-anywhere -- \
    --flake '.#opencloud-pve' \
    --phases install \
    root@opencloud.stfl.dev
#+end_src

** Boot into installed system

1. Detach ISO from VM in Proxmox web UI
2. Reboot VM
3. Remove stale SSH host key: =ssh-keygen -R opencloud.stfl.dev=

* Update

#+begin_src bash
nixos-rebuild \
    --target-host opencloud.stfl.dev \
    --sudo \
    switch \
    --flake ".#opencloud-pve"
#+end_src

* Verify services

#+begin_src bash
curl -k https://opencloud.stfl.dev
curl -k https://office.stfl.dev
#+end_src
